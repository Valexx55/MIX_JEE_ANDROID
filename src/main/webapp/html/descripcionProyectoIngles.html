About us

Openshift: is a free platform for application deployment, behaves like an administrator between infrastructure and software platforms, to facilitate the development of web applications. This allows the deployment of applications Java, JavaScript, Perl, PHP, Python and Ruby resulting in the execution of arbitrary code and the use of Framework. OpenShift supports a wide range of languages, Frameworks, Middleware, data sources and developer tools.
