About us

Openshift: is a free platform for application deployment, behaves like an administrator between infrastructure and software platforms, to facilitate the development of web applications. This allows the deployment of applications Java, JavaScript, Perl, PHP, Python and Ruby resulting in the execution of arbitrary code and the use of Framework. OpenShift supports a wide range of languages, Frameworks, Middleware, data sources and developer tools. Part of this technology we have used for this web project made by the students of the course Information Management Systems (Femxa). This website will help us to know the visual pathologies without medical rigor.Technologies used
Java, Mysql, Html, JavaScript, phpMyAdmin, Openshift

من نحن

هو منبر حر لنشر التطبيقات، يتصرف كمسؤول بين البنية التحتية ومنصات البرمجيات، لتسهيل تطوير تطبيقات الويب. وهذا يسمح للنشر تطبيقات جافا، جافا سكريبت، بيرل، بايثون روبي ويؤدي إلى تنفيذ تعليمات برمجية عشوائية واستخدام الإطار. وهو يدعم مجموعة واسعة من اللغات، والأطر، الوسيطة ومصادر البيانات وأدوات المطورين. بعض من هذه التكنولوجيا التي استخدمت لهذا المشروع على شبكة الإنترنت التي أدلى بها إدارة نظم المعلومات دورة الطلاب. في هذا الموقع سوف تساعدنا على مواجهة الأمراض البصرية دون الصرامة الطبية. التقنيات المستخدمة:

Java, Mysql, Httml, JavaScript, phpMyAdmin, Openshift

Sobre Nosotros

Openshift: es una plataforma gratuita para el despliegue de aplicaciones, se comporta como un administrador entre la infraestructura y las plataformas de software, para facilitar el desarrollo de aplicaciones web. Esta permite el despliege de aplicaciones Java, JavaScript, Perl, PHP, Python y Ruby dando lugar a la ejecución de código arbitrario y al uso de Framework. OpenShift soporta un amplio abanico de lenguajes, Frameworks, Middleware, fuentes de datos y herramientas de desarrollador. Parte de esta tecnologia la hemos utilizado para este proyecto web realizados por los alumnos del curso Sistemas de Gestion de la informacion(Femxa). Esta web nos ayudara a conocer las patologias visuales sin rigor medico. Tecnologias utilizadas
Java, Mysql, Html, JavaScript, phpMyAdmin, Openshift
