About

Visual Pathologies is a project carried out by the students of the course of information management systems taught in Femxa. This website helps us to know the practical visual pathologies and their diagnosis. In addition we can do a test without scientific rigor, to know that can be our visual problem. To carry out this project we have used the following technologies
Java, Mysql, Html, JavaScript, phpMyAdmin, Openshift
